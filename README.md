## Run Locally ğŸƒ

Clone the project (make sure You have [Git](https://git-scm.com/) installed)

```bash
  git clone https://github.com/udborets/portfolio.git
```

```bash

```

Install dependencies (make sure You have [NodeJS](https://nodejs.org/en) installed)

```bash
  npm install
```

```bash
  npm run start
```
